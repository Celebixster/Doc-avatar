# Appa

{% hint style="info" %}
Vous ne subissez aucun dégâts <mark style="color:green;">**PVE**</mark> ( <mark style="color:orange;">**Lave**</mark>/<mark style="color:blue;">**Monstre**</mark>/<mark style="color:green;">**Chute**</mark> )
{% endhint %}

{% hint style="success" %}
Vous recevez l'identité d'<mark style="color:green;">**Aang**</mark> à l'annonce des rôles
{% endhint %}

Vos habiletés dépendent de votre **poids ou de votre nombre de slot remplit** ( l'armure n'est pas comprise )

|                     |                                                                            |                                                                                                                                                            |
| ------------------- | -------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Entre 27 et 36 slot | Vous subissez l'effet <mark style="color:blue;">**Lenteur I**</mark> (20%) | Vous recevez l'effet <mark style="color:blue;">**Résistance I**</mark>.                                                                                    |
| Entre 18 et 27 slot | Vous subissez l'effet <mark style="color:blue;">**Lenteur I**</mark> (10%) | Vous recevez 10% de <mark style="color:blue;">**Résistance**</mark>.                                                                                       |
| Entre 9 et 18 slot  | Vous ne possédez aucun malus                                               | Vous pouvez **en faisant un clique droit et sauter en sneak, **<mark style="color:yellow;">**Voler**</mark> pendant **6 secondes** toutes les 8 minutes.   |
| Entre 0 et 9 slot   | Vous possédez 15% de <mark style="color:yellow;">**Vitesse**</mark>        | Vous pouvez **en faisant un clique droit et sauter en sneak, **<mark style="color:yellow;">**Voler**</mark> pendant **12 secondes** toutes les 12 minutes. |

Une fois en période de <mark style="color:yellow;">**Vol**</mark> vous émettez des **particules blanches et grises autour de vous**.
